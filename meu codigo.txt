Comandos do Git

 -git push - Add o arquivo a nuvem no github

 -git add . - add todos os aquivos pendentes(alterados)

 -git commit -m "nome da nova versão" - estrutura para criação de nova COMITE.

 -git status - verifica se tem algum arquivo que não foi carregado na nuvem

 - git reflog - verifica o historico de atualizações 

 - como navegar entre versões do meu codigo 
git reset --hard (numero da versão que você quer voltar)

 - o que são branches e como criar uma
git branch - indentifica quais branch estão ativas
git branch "nomeDaNovaBranch" - foma de como criar nova branch
git checkout "nomeDaBranch" - comando para mudar para outra branch
