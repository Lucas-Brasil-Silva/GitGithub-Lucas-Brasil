Comandos do Git

 -git push - Add o arquivo a nuvem no github

 -git add . - add todos os aquivos pendentes(alterados)

 -git commit -m "nome da nova versão" - estrutura para criação de nova COMITE.

 -git status - verifica se tem algum arquivo que não foi carregado na nuvem

 - git reflog - verifica o historico de atualizações 

 - como navegar entre versões do meu codigo 
git reset --hard (numero da versão que você quer voltar)

 - o que são branches e como criar uma
git branch - indentifica quais branch estão ativas
git branch "nomeDaNovaBranch" - foma de como criar nova branch
git checkout "nomeDaBranch" - comando para mudar para outra branch
git merge "nomeDaBranch" - comando vai servir para add as alterações de uma branch em outra 
git pull - traz as versoes mais recentes do sistema pra sua maquina 

-- PASSOS QUE VOÊ DEVE SEGRUIR SE ESTIVER TRABALHANDO EM UMA EQUIPE, COMO MEDIDAS DE SEGURANÇA

-git pull da branch principal
-gerar uma nova branch a partir da branch principal
-trabalhar e adicinar novas funcionalidades na nova branch que criou
-finalizar o trabalho na branch temporaria
-git checkout na branch principal
-git pull
-mergiar(unir) o codigo da branch temporario com a branch principal(depois de testar)
-git push da branch principal
